## 🎮 基本操作

### 开始游戏
1. 运行程序后，会看到主菜单
2. 选择一个难度级别（建议从"中等"开始）
3. 看到目标文本后，开始输入
4. 输入第一个字符后自动开始计时

### 游戏中
- **正确的字符**：显示为绿色 ✅
- **错误的字符**：显示为红色 ❌
- **Backspace**：删除上一个字符
- **ESC**：返回菜单/重新开始

### 完成后
- 查看详细统计：速度(WPM)、准确率、用时
- 获得评级：打字大师/优秀/良好
- 选择重新开始或返回菜单

## 💡 第一次玩的建议

1. **选择"简单"或"中等"难度**
   - 简单：短句子，适合热身
   - 中等：技术名言，有一定挑战

2. **不要急于求成**
   - 先追求准确率，再追求速度
   - 准确率 > 95% 后再提速

3. **正确的手指位置**
   - 左手：A S D F（食指在F）
   - 右手：J K L ;（食指在J）
   - 大拇指：空格键

4. **练习时间**
   - 每天15-30分钟
   - 分多次练习比一次长时间更有效

## ❓ 常见问题

### Q: 什么是WPM？
A: Words Per Minute（每分钟单词数），是打字速度的标准衡量单位。一般来说：
- 30-40 WPM：初学者
- 40-60 WPM：中等水平
- 60-80 WPM：良好水平
- 80+ WPM：优秀水平

### Q: 如何提高打字速度？
A: 
1. 使用正确的手指位置（盲打）
2. 不要看键盘，眼睛盯着屏幕
3. 保持节奏，不要时快时慢
4. 每天坚持练习

### Q: Windows上终端版无法运行？
A: Windows需要额外的curses库支持：
```bash
pip install windows-curses
```
程序首次运行时会自动尝试安装。

### Q: 图形版无法运行？
A: 确保已安装pygame：
```bash
pip install pygame
```

### Q: 图形版中文显示乱码或方块？
A: 已修复！最新版本会自动检测并加载系统中文字体。
- Windows: 自动使用微软雅黑/宋体/黑体
- macOS: 自动使用PingFang SC/Heiti SC
- Linux: 自动使用文泉驿/Noto Sans CJK

### Q: 终端版颜色显示异常？
A: 确保你的终端支持颜色。大多数现代终端（GNOME Terminal、iTerm2、Windows Terminal等）都支持。

### Q: 可以添加自己的练习文本吗？
A: 当前版本暂不支持，但你可以编辑源代码中的`TEXTS`字典添加自定义文本。
