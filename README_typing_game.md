# ⌨️ 超级打字练习游戏

一个有趣且互动性强的Python打字练习软件，提供终端版和图形界面版两种选择！

## ✨ 特色功能

### 🎮 互动性强
- **实时反馈**：输入时立即显示正确（绿色）或错误（红色）
- **连击系统**：连续正确输入会累积连击数
- **粒子效果**：完成时有精美的烟花效果（图形版）
- **进度条**：实时显示练习进度

### 🎯 趣味性足
- **4种难度级别**：
  - 简单：短句子，适合初学者
  - 中等：技术名言，适合练习者
  - 困难：长句子，挑战高手
  - 编程挑战：Python代码片段，程序员专属
  
- **评级系统**：根据速度和准确率给出评级
  - 🏆 打字大师：WPM ≥ 80 且准确率 ≥ 95%
  - ⭐ 优秀：WPM ≥ 60 且准确率 ≥ 90%
  - 👍 良好：WPM ≥ 40 且准确率 ≥ 85%
  - 💪 继续加油：继续练习！

- **详细统计**：
  - WPM（每分钟单词数）
  - 准确率百分比
  - 用时统计
  - 错误数统计

## 📦 安装

### 终端版（typing_game.py）

**Linux/macOS:**
使用Python标准库，无需额外安装：
```bash
# 直接运行
python3 typing_game.py
```

**Windows:**
首次运行会自动尝试安装所需库，或手动安装：
```bash
# 安装Windows curses支持
pip install windows-curses

# 运行游戏
python typing_game.py
```

### 图形界面版（typing_game_gui.py）⭐ 推荐

**所有平台（Windows/Linux/macOS）:**
```bash
# 安装依赖
pip install pygame

# 运行游戏
python typing_game_gui.py
```

或者使用requirements文件一键安装所有依赖：
```bash
pip install -r requirements_typing.txt
```

**✨ 新增功能：** 图形版已自动适配中文字体，支持中文正常显示！

## 🎮 使用方法

### 终端版
```bash
python3 typing_game.py
```

**操作说明：**
- 数字键 1-4：选择难度
- 输入文字：开始打字练习
- Backspace：删除字符
- ESC：重新开始当前练习
- Q：退出游戏

### 图形界面版（推荐）
```bash
python3 typing_game_gui.py
```

**操作说明：**
- 鼠标点击：选择难度和按钮
- 输入文字：开始打字练习
- Backspace：删除字符
- ESC：返回菜单/退出游戏

## 📊 游戏玩法

1. **选择难度**：从4个难度级别中选择一个
2. **开始练习**：看到目标文本后开始输入
3. **实时反馈**：
   - 绿色表示输入正确
   - 红色表示输入错误
   - 连续正确会累积连击
4. **查看结果**：完成后查看详细统计和评级
5. **继续练习**：可以重新开始或返回菜单

## 🔧 系统要求

- **Python**: 3.6+
- **操作系统**: Windows/Linux/macOS（已全面支持！）
- **终端版**：

## 💡 提示技巧

1. **提升速度**：
   - 熟悉键盘布局
   - 使用正确的手指位置
   - 保持节奏，不要急于求成

2. **提高准确率**：
   - 放慢速度，确保每个字符正确
   - 专注于屏幕上的文本
   - 使用触觉而非视觉找键

3. **有效练习**：
   - 从简单难度开始
   - 每天坚持练习15-30分钟
   - 逐步提升难度
